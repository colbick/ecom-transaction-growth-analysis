# ecom-transaction-growth-analysis
В этом проекте анализируются транзакции e-commerce: очистка данных, построение ключевых KPI, проводится когортный анализ удержания и RFM-сегментации клиентов.  Проверяюся бизнес-гипотезы и строится модель для прогноза повторной покупки, чтобы сформировать практические рекомендации для роста LTV и приоритизации клиентов.
